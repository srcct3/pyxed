# Basic image editing cli using pillow

- resize Resize image
- rotate Rotate image
- flip Flip image
- crop Crop image
- convert Convert image formats
- metadata Show image metadata
- print Print image as ascii
- canvas Expand image canvas

## Installation

- get [uv](https://github.com/astral-sh/uv)

```sh
git clone https://github.com/srcct3/pyxed.git
cd pyxed
uv tool install .
```

## Usage

```sh
pyxed -h
```
